イベントについて

あなたの好きなゆらめきが、
きっと見つかる。

見ているだけで、わたしたちを穏やかな気持ちにさせてくれる、キャンドルの炎。

アロマの香りや、芯がパチパチと燃える音など、様々な感覚を通して楽しませてくれるのも、キャンドルの魅力です。

YURAGI MARCHÉ（ゆらぎマルシェ）は、『キャンドルと送る、安らぎのある生活の提案』をコンセプトに、全国のキャンドルショップと協力して、キャンドルの魅力を発信するイベントです。

仕事や私生活で忙しい方も、そうでない方も。
今夜だけはキャンドルのゆらめきに、身を任せてみませんか？
